USDA Food Composition Databases API

The USDA Food Composition Databases API allows developers to incorporate nutrient data into their websites and applications. The API provides Food Reports that list the nutrient values for a specified food (e.g. cheddar cheese, potatoes) and Nutrient Reports that return lists of foods and their nutrient values for a given set of nutrients (e.g. protein, iron).
